Install (Chrome / Chromium)

How do I install the extension?
  1. Download the build from the Download button.
  2. Unzip the folder on your PC.
  3. Open chrome://extensions
  4. Enable Developer mode (top right).
  5. Click Load unpacked and select the unzipped folder.
  6. Pin the extension (optional) and click it to launch.
How do I update it?
Replace the files in the same unpacked folder, then press Reload on the extension card in chrome://extensions.
